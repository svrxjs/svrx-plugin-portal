# svrx-plugin-portal

The portal plugin for svrx. Provide intranet penetration

```bash
svrx --portal
```

| Arguments | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| remoteHost | string | igame.163.com | server host |
| subdomain | string | | subdomain |
| fallback | object | | proxy configuration |
